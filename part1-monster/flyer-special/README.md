# 飞行的怪物

MC里会飞的生物不少，但是这些生物的底层实现差别不小，而且个人觉得这方面的实现比较混乱。根据使用的是`FlyingMoveControl`还是自定义的`MoveControl`可以把飞行生物分为两类。  

不过可以观察得到一个规律，飞行方式与现实中鸟类相似的，大多使用了`FlyingMoveControl`，差别大的则大多不使用`FlyingMoveControl`。具体可见1.2.4.1节。  
  
1.2.4里介绍的，主要是飞行的怪物，由于这些怪物的飞行方式与鸟类差别较大，所以一般不使用`FlyingMoveControl`。  

*由于凋灵是Boss，本部分不讨论凋灵，但是要注意，凋灵虽然**看上去**飞行方式“与鸟类不太一样”，但是凋灵**使用**了`FlyingMoveControl`。*
